# author.contact

Contact information, such as an email address, GitHub or Twitter

> **Note for authors of public packages**: This value will appear in your package metadata and GitHub repository

## Default Value

`https://github.com/` + [`github.username`](/docs/cli-reference/github-username/)

## Example Update

```
$ hats config author.contact sergio.kitchens@example.com
```
