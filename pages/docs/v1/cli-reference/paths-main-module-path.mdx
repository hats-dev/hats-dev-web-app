# paths.main-module-path

The path to the JavaScript file serving as the entry point for your module

## Default Value

`dist/index.js`

## Description

The value of this config is assigned to the `main` property in your `package.json` file.

You should **only** change this value in tandem with an update to your [`paths.build-dir-path`](/docs/cli-reference/paths-build-dir-path) config.

## Example Update

```
$ hats config paths.main-module-path "lib/index.js"
```
