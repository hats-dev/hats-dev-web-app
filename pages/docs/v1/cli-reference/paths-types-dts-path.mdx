# paths.types-dts-path

The path to the declaration file for your module

## Default Value

`dist/index.d.ts`

## Description

The value of this config is assigned to the `types` property in your `package.json` file.

You should **only** change this value in tandem with an update to your [`paths.build-dir-path`](/docs/v1/cli-reference/paths-build-dir-path) config.

## Example Update

```
$ hats config paths.types-dts-path "lib/index.d.ts"
```
