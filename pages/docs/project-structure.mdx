# Project Structure

The HaTs project structure boosts productivity, promotes good habits, and it just works.

## Directory Architecture

The default layout for HaTs project files reflects of the baseline development workflow.

```bash
.
├── .eslintignore
├── .eslintrc.json
├── .gitignore
├── .prettierignore
├── LICENSE
├── README.md
├── package-lock.json
├── package.json
├── src
│   ├── index.ts
│   └── index.test.ts
├── tests
│   └── global.test.ts
├── tsconfig-build.json
└── tsconfig.json
```
